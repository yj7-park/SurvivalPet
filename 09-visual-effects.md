# 설계 09 — 시각 이펙트 (물 · 나무 · 캐릭터 동작)

> **전제 조건**: 08 단계 완료 상태.
> 맵 렌더링, 캐릭터 이동, 상호작용 시스템, 파티클 기반이 구현되어 있다고 가정한다.

---

## 1. 이번 단계 목표

1. 물 타일 애니메이션 (물결, 반짝임)
2. 캐릭터가 물에서 이동 시 파문/물튀김
3. 나무 바람 흔들림 / 벌목 쓰러짐 연출
4. 캐릭터 동작별 이펙트 (이동, 채집, 전투, 수면, 광란 등)

---

## 2. 물 타일 이펙트

### 2-1. 물결 애니메이션

물 타일 위에 반투명 레이어를 오버레이하여 물결을 표현.
타일 단위가 아닌 **전체 물 영역을 하나의 셰이더/그래픽으로 처리**.

```
구현 방식: Phaser RenderTexture + 시간 기반 사인 오프셋

각 물 타일마다 y오프셋을 시간 함수로 계산:
  offsetY = sin(time * 1.5 + tileX * 0.8) * 1.5  (px)

반짝임 하이라이트:
  매 2.5~4초 간격으로 물 타일 일부에
  흰 반투명 타원 (alpha 0.3~0.5) 이 나타났다 사라짐
  지속 시간: 0.6초, 페이드 인/아웃
```

### 2-2. 날씨 연동

| 날씨 | 물 이펙트 변화 |
|------|---------------|
| 맑음 | 기본 (반짝임 강조) |
| 흐림 | 반짝임 제거, 물결 느리게 |
| 비 | 물 표면에 빗방울 파문 파티클 추가 |
| 폭풍 | 파문 빠르고 크게, 흰 물거품 파티클 |

**빗방울 파문 파티클 (비 날씨)**
```
texture: 얇은 원형 링 (코드 생성, 16×16)
emitZone: 물 타일 전체 영역
quantity: 2~3/초/타일
lifespan: 500ms
scale: { start: 0.2, end: 1.2 }
alpha: { start: 0.6, end: 0 }
tint: #aaccff
```

### 2-3. 캐릭터 입수 / 이동 파문

캐릭터가 물 타일에 진입하거나 물 위를 걷는 동안:

**입수 순간 (물 타일 처음 진입)**
```
파티클: 흰/파란 작은 물방울 8개, 방사형 분출
  speed: 40~80px/s, 방사형
  lifespan: 400ms
  scale: 0.3~0.6
  tint: [#ffffff, #aaddff]
  gravity: 200 (아래로 당겨짐)
```

**물 위 이동 중 (걸음마다)**
```
캐릭터 발 위치에 원형 파문 생성 (2걸음마다 1회)
  texture: 원형 링
  scale: { start: 0.3, end: 1.0 }
  alpha: { start: 0.5, end: 0 }
  lifespan: 600ms
```

## 3. 나무 이펙트

### 3-1. 바람 흔들림 (상시)

나무 오브젝트 전체에 주기적인 좌우 흔들림 적용.

```
흔들림 공식:
  rotation = sin(time * windSpeed + treeIndex * 1.3) * maxAngle

  맑음/흐림:  windSpeed=0.8,  maxAngle=1.5°
  비:         windSpeed=1.5,  maxAngle=3.0°
  폭풍:       windSpeed=3.0,  maxAngle=6.0°
  블리자드:   windSpeed=4.0,  maxAngle=8.0°

각 나무마다 treeIndex가 달라 모두 같이 흔들리지 않고 자연스럽게 차이남.
흔들림 기준점: 나무 하단 중앙 (줄기 뿌리)
```

### 3-2. 벌목 중 이펙트 (프로그레스 진행 중)

```
도끼 타격마다 (프로그레스 20%마다):
  1. 나무가 타격 방향 반대로 0.2초간 흔들림 (rotation +5°)
  2. 나무 파편 파티클 3~5개 방출
       texture: 작은 갈색 사각형 (4×4 ~ 6×6)
       speed: 60~120px/s, 부채꼴 방향
       lifespan: 500ms
       tint: [#8b5e2a, #c8884a]
       gravity: 300
  3. 타격 효과음 위치에 흰 별표 이펙트 0.1초 표시
```

### 3-3. 벌목 완료 — 나무 쓰러짐

```
1. 나무가 쓰�러지는 방향: 플레이어 반대편
2. rotation 애니메이션: 0° → 90° (0.6초, easeIn 곡선)
3. 쓰러짐 완료 시:
   - 나무 오브젝트 제거
   - 충격 파티클: 먼지 + 나무 파편 15~20개 방출
       먼지: 회색 큰 원형, scale 1→3, alpha 0.4→0, 800ms
       파편: 갈색 사각형, 방사형, gravity 400, 600ms
   - 지면 진동: camera.shake(200, 0.005)
4. 목재 아이템 드롭 위치에 반짝임 이펙트 표시
```

## 4. 캐릭터 동작 이펙트

### 4-1. 이동

**일반 지형 (흙)**
```
걸음 2보마다 먼지 파티클 1개
  texture: 작은 회색 원형 (6×6)
  speed: 20px/s, 아래+측면 방향
  lifespan: 300ms
  alpha: { start: 0.4, end: 0 }
  scale: { start: 0.5, end: 1.0 }
```

**암반 위 이동**
```
먼지 대신 작은 회색 파편
  tint: #909090
  lifespan: 200ms
```

**물 위 이동**: 2-3절 참조

### 4-2. 벌목 / 채굴 / 낚시 (상호작용 중)

**벌목 (프로그레스 중)**
- 캐릭터 팔이 도끼 휘두르는 방향으로 0.15초 기울어짐 (rotation ±15°)
- 타격 시 캐릭터 앞에 흰 별표 이펙트

**채굴 (프로그레스 중)**
```
타격마다:
  암석 파편 파티클 3~4개 (회색, 작은 사각형)
  camera.shake(80, 0.003)  ← 약한 진동
  캐릭터 팔이 망치 내려치는 동작 (rotation +20° → 0°, 0.2초)
```

**낚시 (프로그레스 중)**
```
낚싯줄 표현:
  캐릭터 손 → 인접 물 타일 방향으로 얇은 선 (1px, #d4c08a)
  찌 표현: 물 위에 빨간 작은 원 (4×4)
  찌가 물결에 따라 위아래 bobbing (sin 함수, 1초 주기)
낚시 성공 순간:
  찌가 물속으로 빠르게 내려가는 연출 (0.3초)
  물 파문 파티클 방출
```

### 4-3. 전투

**근접 공격 (칼 / 맨손)**
```
공격 방향으로 호(arc) 이펙트:
  흰/노랑 반투명 부채꼴 (120° 범위, 캐릭터 앞 1.5타일)
  lifespan: 150ms
  alpha: { start: 0.7, end: 0 }
캐릭터 몸체: 0.1초 공격 방향 기울어짐
```

**원거리 공격 (활)**
```
시위 당기는 모션: 0.3초간 캐릭터 팔 뒤로 당김 표현
발사 시: 활 반동 (앞으로 0.1초 vibration)
화살 비행 중: 화살 뒤에 짧은 흰 잔상 (trail, 3px, 150ms)
```

**피격**
```
캐릭터 스프라이트 0.2초 빨간 플래시 (tint #ff4444)
피격 방향 반대로 1.5타일 넉백 (0.15초, easeOut)
빨간 피 파티클 5~8개
  speed: 50~100px/s, 방사형
  lifespan: 400ms
  tint: #cc0000
  gravity: 200
```

**회피 성공 (Dodge)**
```
캐릭터가 피격 방향 반대로 0.5타일 빠르게 이동 (0.1초)
하늘색 "DODGE!" 텍스트 팝업 (위로 1초 이동 후 페이드)
잔상 이펙트: 캐릭터 위치에 반투명 복사본 0.3초간 페이드 아웃
```

### 4-4. 수면 상태 (피로 0 강제 수면 / 침대 수면)

```
캐릭터 위에 ZZZ 텍스트 파티클:
  "Z" → "ZZ" → "ZZZ" 순으로 1.5초 간격 생성
  각 Z: 위로 천천히 이동 (40px/s), 점점 작아지며 페이드 아웃
  lifespan: 2500ms
  tint: #aaccff
캐릭터 스프라이트: 반투명 처리 (alpha 0.6), 약간 기울어짐 표현
```

### 4-5. 광란 모드 (행동 수치 0)

```
캐릭터 눈 부분: 빨간 글로우 이펙트
  Phaser postFX glow 또는 캐릭터 위에 작은 빨간 점 깜빡임

캐릭터 주변 빨간 오라:
  캐릭터 중심에서 방사형 빨간 파티클
  quantity: 3/frame
  speed: 20~40px/s
  lifespan: 600ms
  tint: #cc0000
  alpha: { start: 0.6, end: 0 }
  blendMode: ADD

캐릭터 이동 시 빨간 잔상:
  0.1초마다 현재 위치에 캐릭터 반투명 복사본 생성
  tint: #ff0000, alpha 0.3 → 0 (0.4초)
```

### 4-6. 아이템 획득

```
획득 순간 아이템 위치에서:
  노란 반짝임 파티클 6개 방출 (별 모양)
  speed: 30~60px/s, 방사형
  lifespan: 400ms
  tint: #ffdd00
  blendMode: ADD

인벤토리 슬롯 해당 칸: 0.3초 노랑 하이라이트 표시
```

### 4-7. 사망

```
1. 0.5초 동안 캐릭터 스프라이트 회전 (0° → 90°, easeIn)
2. 회색 톤으로 변색 (tint #888888)
3. 빨간 파티클 폭발 (15~20개)
4. 1초 후 페이드 아웃
5. 아이템 드롭 → 각 아이템에 반짝임 이펙트
```

## 5. 이펙트 레이어 순서 (Z-order)

```
레이어 0: 타일 (흙/물/암반)
레이어 1: 물 이펙트 (물결, 파문)
레이어 2: 바닥 아이템
레이어 3: 구조물 하단 (벽, 침대 등)
레이어 4: 캐릭터 / 동물 / 적군
레이어 5: 나무 오브젝트
레이어 6: 지면 파티클 (먼지, 파편)
레이어 7: 캐릭터 이펙트 (잔상, 오라, ZZZ)
레이어 8: 공중 파티클 (낙엽, 눈, 비)
레이어 9: UI 오버레이 (HP바, 프로그레스바, 팝업 텍스트)
레이어 10: 화면 오버레이 (낮밤, 날씨 오버레이, 빨간 테두리)
```
