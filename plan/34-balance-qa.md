## 9. QA 체크리스트

### 9-1. 핵심 루프

- [ ] 새 게임 생성 → 캐릭터 생성 → 게임 진입 → 타이틀 복귀 흐름 정상
- [ ] 저장 슬롯 3개 독립 저장·불러오기 정상
- [ ] 자동 저장 5분 주기 작동 확인
- [ ] 사망 시 인벤토리 드랍 + 슬롯 삭제 정상

### 9-2. 생존 수치

- [ ] 허기 자연 감소 속도 (STR별 3가지 캐릭터로 검증)
- [ ] 피로 자연 감소 속도 (CON별 검증)
- [ ] 행동 수치 감소 + 광란 30초 → 종료 후 25 회복
- [ ] 허기 0 → 최대HP 75초마다 −5 누적 확인
- [ ] 피로 0 → 강제 수면 → 피로 40 이상 기상 확인

### 9-3. 전투

- [ ] 무기별 데미지·공격속도 정상
- [ ] 방어도 적용 (rawDmg − defense, 최소 1)
- [ ] 방패 막기 확률 샘플링 (100회 공격, 15% ± 5% 범위 내)
- [ ] 적 처치 시 드랍 + XP 지급 정상
- [ ] 광란 중 자동 공격 대상 우선순위 (적 > 플레이어 > 건설물)

### 9-4. 건설 & 내구도

- [ ] 모든 건설물 재료·시간 정상
- [ ] 내구도 노후화 (목재 1시간에 약 −0.07 확인)
- [ ] 지붕 건설 → 실내 판정 → 수면 회복 ×1.2 적용 확인
- [ ] 문 E키 열기/닫기 + 충돌 토글 정상
- [ ] 구조물 파괴 → 재료 50% 드랍 확인

### 9-5. 요리·제작·숙련도

- [ ] 조리대 요리 → 허기 회복·HP 회복 정상
- [ ] 식중독 발생률 (날고기 100회 섭취, 35% ± 8% 범위 내)
- [ ] 숙련도 레벨업 XP 누적·해금 확인 (각 숙련도 Lv.2까지)
- [ ] 연구 완료 → 제작 해금 정상
- [ ] 레시피/도면 아이템 학습 → 요리/제작 메뉴 해금 정상

### 9-6. 맵 & 이동

- [ ] Seed 동일 시 동일 맵 생성 확인 (3회 반복)
- [ ] 맵 전환 4방향 정상 (도착 좌표 반대편 끝 확인)
- [ ] 세계 끝(mapX/Y 0,9 경계) 이동 차단
- [ ] 나무 벌목 → 10분 후 재생 확인
- [ ] 암반 채굴 → 재생 없음 확인

### 9-7. 야간 & 조명

- [ ] 22:00 암흑 진입, 06:00 밝아짐 타이밍
- [ ] 횃불 들기 → 160px 광원 확인
- [ ] 횃불 10분 수명 소진 → 경고 → 소멸
- [ ] 실내(지붕 아래) 암흑 미적용 확인

### 9-8. 멀티플레이

- [ ] 같은 Seed 두 탭 → 동일 맵·동일 건설물 표시
- [ ] 위치 동기화 (100ms 간격, 보간 부드러움)
- [ ] 건설 이벤트 실시간 전파 (A가 지은 벽을 B가 즉시 볼 수 있음)
- [ ] 광란 플레이어 빨간 오라 상대방에게 표시
- [ ] 채팅 메시지 상호 수신·말풍선 표시
- [ ] 탭 강제 종료 → 10초 내 online: false 처리

### 9-9. 저장·불러오기

- [ ] 게임 종료 후 재시작 → 동일 위치·인벤토리·숙련도 복원
- [ ] 맵 전환 시 자동 저장 확인
- [ ] 수면 기상 시 자동 저장 확인
- [ ] 버전 마이그레이션 (구버전 SaveData 로드 시 기본값 삽입)

### 9-10. 성능

- [ ] 초기 로딩 ≤ 3초 (Chrome DevTools Network throttle 없음)
- [ ] 맵 전환 ≤ 0.5초 (캐시 히트 기준)
- [ ] 60fps 유지 (플레이어 4명 + 적 10마리, Chrome Performance 탭)
- [ ] 30분 장시간 플레이 후 메모리 ≤ 300MB

## 10. 밸런스 조정 가이드

조정이 필요할 때 수정 우선 파일:

| 항목 | 수정 파일 |
|------|---------|
| 능력치·파생 스탯 | `src/systems/CharacterStats.ts` |
| 무기·방어구 수치 | `src/config/equipment.ts` |
| 적 스탯 | `src/config/enemies.ts` |
| 음식 효과 | `src/config/foods.ts` |
| 건설 재료·내구도 | `src/config/buildings.ts` |
| 숙련도 XP·해금 | `src/config/recipes.ts`, `ProficiencySystem.ts` |
| 생존 수치 감소 속도 | `src/systems/HungerSystem.ts`, `SleepSystem.ts`, `ActionSystem.ts` |
| 조명 반경·수명 | `src/systems/LightSystem.ts` |

수치 변경 후 반드시 위 QA 체크리스트 해당 항목 재검증.
