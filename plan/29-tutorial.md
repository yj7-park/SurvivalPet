# 설계 29 — 튜토리얼 & 신규 플레이어 가이드

> **전제 조건**: 01~28 단계 완료 상태.
> TitleScene, GameScene, HUD, 모든 상호작용 시스템이 구현되어 있다고 가정한다.

## 1. 이번 단계 목표

1. **첫 플레이 튜토리얼** — 신규 플레이어에게 핵심 조작 순서대로 안내
2. **스텝 기반 진행** — 조건 달성 시 자동으로 다음 단계 진행
3. **스킵 가능** — 경험자는 건너뛸 수 있음
4. **조작 도움말(Cheatsheet)** — H 키로 언제든 조작 가이드 열람
5. **툴팁 강조** — 튜토리얼 중 해당 UI 요소 하이라이트

```typescript
// 캐릭터 생성 완료 → 게임 진입 시

```

## 3. 튜토리얼 스텝 목록

총 10단계, 각 단계는 **조건 달성 시 자동 진행**.

### Step 1 — 이동

```
[안내 텍스트]
"방향키(↑↓←→)로 캐릭터를 이동해보세요."

[하이라이트] 없음 (화면 전체 활성)
[달성 조건] 플레이어가 32px 이상 이동
[스킵] 5초 대기 후 [건너뛰기] 버튼 표시
```

### Step 2 — 나무 벌목

```
[안내 텍스트]
"나무를 클릭해 목재를 얻으세요.
 가까이 이동하면 자동으로 벌목을 시작합니다."

[하이라이트] 가장 가까운 나무 오브젝트에 노란 테두리 + 화살표
[달성 조건] 목재 1개 이상 획득
```

### Step 3 — 인벤토리 확인

```
[안내 텍스트]
"V 키를 눌러 인벤토리를 열어보세요."

[하이라이트] HUD의 인벤토리 단축키 힌트 강조
[달성 조건] 인벤토리 패널 1회 이상 열기
```

### Step 4 — 암반 채굴

```
[안내 텍스트]
"암반(회색 바위)을 클릭해 석재를 채굴하세요.
 암반은 한 번 채굴하면 재생되지 않습니다."

[하이라이트] 가장 가까운 암반 타일
[달성 조건] 암석 1개 이상 획득
```

### Step 5 — 건설

```
[안내 텍스트]
"B 키를 눌러 건설 패널을 열고
 목재 벽을 하나 세워보세요."

[하이라이트] B 키 힌트 + 건설 패널 버튼
[달성 조건] 구조물 1개 이상 건설 완료
```

### Step 6 — 작업대 제작

```
[안내 텍스트]
"작업대를 건설한 뒤 클릭해 낚싯대를 만들어보세요.
 낚싯대가 있으면 낚시 성공률이 오릅니다."

[하이라이트] 없음 (자유 행동)
[달성 조건] 작업대 건설 + 아이템 1개 이상 제작
[선택] 작업대 미건설 시 힌트: "먼저 작업대를 건설하세요 (목재 ×10)"
```

### Step 7 — 낚시

```
[안내 텍스트]
"물 타일 근처로 이동해 물을 클릭하면 낚시를 시작합니다.
 허기가 줄어들면 생선을 구워 먹어야 합니다."

[하이라이트] 가장 가까운 물 타일
[달성 조건] 낚시 1회 완료 (성공·실패 무관)
```

### Step 8 — 요리 & 식사

```
[안내 텍스트]
"조리대를 건설하고 물고기를 구워보세요.
 날것을 먹으면 식중독에 걸릴 수 있습니다."

[하이라이트] 없음
[달성 조건] 음식 아이템 1회 섭취
```

### Step 9 — 수면

```
[안내 텍스트]
"침대를 건설하고 자면 피로를 회복할 수 있습니다.
 피로가 0이 되면 그 자리에 쓰러집니다!"

[하이라이트] 피로 수치 HUD 게이지 강조
[달성 조건] 수면 1회 완료 (기상까지)
```

### Step 10 — 완료

```
[안내 텍스트]
"기본 조작을 모두 익혔습니다! 🎉
 H 키를 누르면 언제든 조작 가이드를 볼 수 있습니다.
 살아남으세요!"

[하이라이트] 없음
[달성 조건] 확인 버튼 클릭
→ localStorage['sv_tutorial_done'] = '1' 저장
→ 튜토리얼 종료
```

## 4. 튜토리얼 UI 구조

### 4-1. 안내 텍스트 박스

화면 하단 중앙 고정:

```
┌─────────────────────────────────────────────┐
│  💡 나무를 클릭해 목재를 얻으세요.           │
│     가까이 이동하면 자동으로 벌목을 시작합니다. │
│                                             │
│  (2/10)          [건너뛰기]   [다음 →]     │
└─────────────────────────────────────────────┘
```

- 배경: 반투명 검정 (`rgba(0,0,0,0.75)`)
- 폰트: 14px monospace
- `[다음 →]`: 달성 조건 충족 시 활성화 (미충족 시 회색)
- `[건너뛰기]`: 전체 튜토리얼 스킵

### 4-2. 하이라이트 오버레이

특정 오브젝트 강조 시:
```typescript
// 해당 오브젝트 외 화면 전체 어둡게 (알파 0.5 검정)
// 오브젝트 주변 노란 테두리 펄스 (width 2px, 1초 주기)
// 화살표 스프라이트 (↓ 16×16) 오브젝트 상단 +8px에서 위아래 부유
```

### 4-3. 진행률 표시

안내 박스 좌측 하단 `(2/10)` 형태.

## 6. TutorialSystem 클래스

```typescript
export class TutorialSystem {
  private currentStep: number = 0;
  private steps: TutorialStep[];
  private active: boolean = false;
  private highlightTarget: Phaser.GameObjects.GameObject | null = null;

  start(): void
  skip(): void

  // 매 프레임: 달성 조건 체크
  update(gameState: GameState): void

  // 외부에서 이벤트 통지
  onEvent(event: TutorialEvent): void

  private advance(): void
  private complete(): void
  private showStep(step: TutorialStep): void
  private highlightObject(obj: Phaser.GameObjects.GameObject | null): void
}

interface TutorialStep {
  id: string;
  message: string[];                    // 여러 줄 텍스트
  highlight?: 'nearest_tree' | 'nearest_rock' | 'nearest_water'
            | 'hud_fatigue' | 'key_B' | 'key_V';
  condition: (state: GameState) => boolean;
}

type TutorialEvent =
  | 'inventory_opened' | 'building_built' | 'item_crafted'
  | 'fish_completed'   | 'food_eaten'     | 'sleep_completed';
```

## 8. 확정 규칙

- 튜토리얼은 **게임플레이를 막지 않음** — 자유롭게 이동·행동하면서 조건 달성
- 각 스텝은 순서대로 진행 (이전 달성 없이 다음 스텝 불가)
- [건너뛰기] 클릭 시 전체 스킵, `sv_tutorial_done` 저장 (재표시 안 함)
- 멀티플레이 중 튜토리얼 진행 가능 — 다른 플레이어에게는 튜토리얼 UI 표시 안 됨
- 튜토리얼 중 사망 시 튜토리얼 상태 초기화 (다음 게임 시작 시 처음부터)
- 조작 가이드(H 키)는 튜토리얼 완료 여부와 무관하게 항상 사용 가능
